# readme


/* 操作方法 */

# 十字キー, アナログパッド
    :カーソルの移動(Play with Gamepadのとき)

# マウス左, ゲームパッドボタン2, Zキー
    :ショット

# マウス右, ゲームパッドボタン3, Xキー
    :貫通ショット(superShot)の開始(スコアを5000消費します)


/* キーコマンド集 */

# "Esc": タイトル画面に戻ります
# "F5": ゲームを終了します

# "i": スコアデータのインポート(読み込み)。タイトル画面のみ
# "e": スコアデータのエクスポート(書き込み)。タイトル画面のみ

   +  +- score.json /*スコアデータ。バックアップはこれを。*/
